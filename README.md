
# Scientific Calculator

A fully functional scientific calculator built using **HTML**, **CSS**, and **JavaScript**.  
It supports basic arithmetic operations as well as scientific functions like trigonometry, logarithms, square root, powers, and factorial.

## Features

- Basic arithmetic operations: addition, subtraction, multiplication, and division.
- Scientific functions: sin, cos, tan, log, ln, sqrt, power, square, and factorial.
- Constants like π and e.
- Parentheses support for advanced expressions.
- Clear and delete buttons.
- Responsive and clean UI.
- Keyboard and mouse-friendly interaction.

## Tech Stack

- HTML5
- CSS3
- JavaScript

## How to Run

1. Download or clone the project.
2. Open the folder in **VS Code**.
3. Open `index.html` in your browser.
4. Or use **Live Server** for a better development experience.

## Usage

- Click the number buttons to enter values.
- Use the operator buttons for calculations.
- Use scientific buttons for advanced math operations.
- Press `=` to get the result.
- Press `AC` to clear everything.
